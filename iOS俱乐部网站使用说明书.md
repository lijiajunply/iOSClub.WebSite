# iOS俱乐部网站管理系统 - 使用说明书

## 1. 系统概述

### 1.1 系统简介
iOS俱乐部网站管理系统是西安建筑科技大学iOS Club为了更好地展示俱乐部信息、管理成员、发布活动和文章而开发的网站平台。该系统采用现代化技术栈，具有界面美观、功能全面、操作便捷等特点。

### 1.2 主要功能
- 俱乐部信息展示
- 活动与项目发布
- 文章与博客系统
- 成员管理与交流
- 后台管理功能

### 1.3 适用对象
- 俱乐部成员
- 学校师生
- 其他组织和个人

## 2. 快速开始

### 2.1 访问网站
- **网址**：[网站域名，如：https://iosclub.xauat.edu.cn]
- **浏览器要求**：建议使用Chrome 90+、Firefox 88+、Safari 14+或Edge 90+浏览器
- **设备要求**：支持桌面端、平板和移动端设备

### 2.2 用户注册
1. 在网站首页点击右上角的「注册」按钮
2. 进入注册页面，填写以下信息：
   - 用户名
   - 邮箱
   - 密码
   - 验证码
3. 阅读并同意用户协议
4. 点击「注册」按钮完成注册
5. 系统将发送验证邮件到您的邮箱，请点击邮件中的链接完成邮箱验证

### 2.3 用户登录
1. 在网站首页点击右上角的「登录」按钮
2. 进入登录页面，输入您的用户名/邮箱和密码
3. 点击「登录」按钮完成登录
4. 登录成功后，您将被重定向到网站首页

### 2.4 忘记密码
1. 在登录页面点击「忘记密码」链接
2. 进入密码找回页面，输入您的注册邮箱
3. 点击「发送验证码」按钮，系统将发送验证码到您的邮箱
4. 输入收到的验证码，设置新密码
5. 点击「确认重置」按钮完成密码重置

## 3. 网站首页使用

### 3.1 首页布局
- **顶部导航栏**：包含网站Logo、主要导航链接和用户信息
- **轮播图区域**：展示俱乐部最新活动和重要通知
- **功能模块区**：快速访问俱乐部活动、项目、文章等功能
- **成员风采区**：展示俱乐部成员信息
- **底部信息区**：包含联系方式、版权信息等

### 3.2 导航菜单
- **首页**：返回网站首页
- **活动**：查看俱乐部活动信息
- **项目**：浏览俱乐部项目
- **文章**：阅读俱乐部博客文章
- **关于我们**：了解俱乐部基本信息
- **登录/注册**：用户登录和注册入口

### 3.3 快速操作
- **搜索功能**：在顶部导航栏的搜索框中输入关键词，搜索网站内容
- **主题切换**：点击右上角的主题切换按钮，切换浅色/深色主题
- **移动端适配**：在移动设备上，点击左上角的菜单按钮展开导航菜单

## 4. 俱乐部活动模块

### 4.1 活动列表
1. 在导航菜单中点击「活动」
2. 进入活动列表页面，浏览所有活动
3. 可以通过分类筛选或搜索功能查找特定活动

### 4.2 活动详情
1. 在活动列表中点击任意活动卡片
2. 进入活动详情页面，查看活动的详细信息
3. 可以在详情页下方留言或分享活动

### 4.3 项目展示
1. 在导航菜单中点击「项目」
2. 进入项目列表页面，浏览所有项目
3. 点击项目卡片查看项目详情
4. 项目详情页包含项目介绍、成员信息和成果展示

### 4.4 历史记录
1. 在导航菜单中点击「历史」
2. 进入历史记录页面，查看俱乐部的发展历程
3. 可以按年份筛选查看不同时期的历史

## 5. 文章与博客系统

### 5.1 文章列表
1. 在导航菜单中点击「文章」
2. 进入文章列表页面，浏览所有文章
3. 可以通过分类、标签或搜索功能查找特定文章

### 5.2 文章阅读
1. 在文章列表中点击任意文章标题
2. 进入文章详情页面，阅读文章内容
3. 文章支持Markdown格式，包含文字、图片、代码等内容
4. 可以在文章底部评论或分享

### 5.3 关于我们
1. 在导航菜单中点击「关于我们」
2. 查看俱乐部的详细介绍、宗旨和发展目标

### 5.4 社团结构
1. 在导航菜单中点击「社团结构」
2. 查看俱乐部的组织架构、部门设置和成员信息

## 6. 成员中心

### 6.1 个人资料
1. 登录后，点击右上角的用户名
2. 选择「个人资料」
3. 进入个人资料页面，查看和编辑个人信息

2. 选择「我的活动」

### 6.3 我的文章

2. 选择「我的文章」

4. 可以编辑或删除自己的文章

## 7. 后台管理系统（管理员使用）

### 7.1 登录后台
1. 确保您拥有管理员权限
2. 登录网站后，点击右上角的用户名
3. 选择「管理中心」
4. 进入后台管理页面

### 7.2 内容管理

#### 7.2.1 文章管理
1. 在后台左侧菜单中点击「文章管理」
2. 进入文章列表页面，可以查看、编辑、删除文章
3. 点击「新增文章」按钮，进入文章编辑页面
4. 填写文章标题、内容、分类等信息
5. 点击「保存」按钮发布文章

#### 7.2.2 分类管理
1. 在后台左侧菜单中点击「分类管理」
2. 进入分类列表页面，可以查看、编辑、删除分类
3. 点击「新增分类」按钮，创建新的分类

### 7.3 活动与项目管理

#### 7.3.1 活动管理
1. 在后台左侧菜单中点击「活动管理」
2. 进入活动列表页面，可以查看、编辑、删除活动
3. 点击「新增活动」按钮，创建新的活动

#### 7.3.2 项目管理
1. 在后台左侧菜单中点击「项目管理」
2. 进入项目列表页面，可以查看、编辑、删除项目
3. 点击「新增项目」按钮，创建新的项目

### 7.4 成员管理
1. 在后台左侧菜单中点击「成员管理」
2. 进入成员列表页面，可以查看、编辑、删除成员
3. 可以设置成员角色和权限

### 7.5 系统设置
1. 在后台左侧菜单中点击「系统设置」
2. 可以设置网站基本信息、主题、SEO等
3. 可以查看系统日志和统计数据

## 8. 常见问题与解决方案

### 8.1 登录问题
- **问题**：无法登录系统
- **解决方案**：
  1. 检查用户名和密码是否正确
  2. 检查网络连接是否正常
  3. 如果忘记密码，点击「忘记密码」重置密码
  4. 联系管理员协助解决

### 8.2 功能访问问题
- **问题**：无法访问某些功能
- **解决方案**：
  1. 检查是否已登录系统
  2. 检查是否拥有相应的权限
  3. 刷新页面或清除浏览器缓存后重试

### 8.3 内容发布问题
- **问题**：无法发布文章或活动
- **解决方案**：
  1. 检查内容是否符合要求（如字数限制、格式要求）
  2. 检查网络连接是否正常
  3. 联系管理员检查系统状态

### 8.4 浏览器兼容性问题
- **问题**：页面显示异常或功能无法使用
- **解决方案**：
  1. 确保使用推荐的浏览器（Chrome、Firefox、Safari、Edge）
  2. 更新浏览器到最新版本
  3. 清除浏览器缓存和Cookie后重试
